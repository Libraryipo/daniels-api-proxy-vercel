# Daniels Orchestral Music API Proxy

This is a Vercel-hosted proxy server for the Daniels Orchestral Music API, designed for Monday.com integrations.

## Endpoints
- **POST** `/api/search`
- **POST** `/api/fetch`

## Deployment
1. Push this repo to GitHub.
2. Import into [Vercel](https://vercel.com).
3. Vercel will auto-detect `/api` serverless functions.
